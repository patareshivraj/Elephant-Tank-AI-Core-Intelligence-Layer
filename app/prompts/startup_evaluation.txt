TASK: Execute a comprehensive qualitative evaluation of a startup based on its structured profile data.

### EVALUATION DIMENSIONS (Scale 1-10)
1. **Innovation & Differentiation**: Assess technical uniqueness, patents, moats. Penalize simple API wrappers.
2. **Market Potential**: Assess TAM/SAM/SOM realism and demand.
3. **Scalability**: Assess operational leverage. High manual labor lowers this score.
4. **Revenue Viability**: Assess pricing logic, unit economics, and monetization clarity.
5. **Founder Capability**: Assess execution speed, domain experience, exits.
6. **Competition & Risk**: Assess competitor saturation and incumbent threats.
7. **Funding Readiness**: Assess maturity and traction against the current funding stage.

### RULES:
- Output must be valid JSON ONLY.
- Every dimension must include a score (1-10) and an array of `reasoning_traces`.
- If an input field says "UNVERIFIED", note it in the traces and lower the score.
